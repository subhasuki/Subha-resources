LoremIpsum

An extension for Sublime Text 2 and 3! It allows you to Insert Lorem Ipsum in the editor via menu items or keyboard shortcut. Select how much text you want from the menu item in Edit->Text->Lorem Ipsum or on the right click menu in Lorem Ipsum. Just press the shortcut key (Alt+Shift+L) to add Lorem Ipsum text. Keep pressing to add more. Alternatively, you can also specify the number of paragraphs, by writing a number, and pressing the shortcut key. Lorem Ipsum will check if there is a number directly preceding the selection range, and replace it with that many paragraphs of lorem ipsum. Addind a decimal point, defines the approximate number of words per paragraph.
